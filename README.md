# 🏠 CelestiAR: AR Interior Design App

## 📌 Project Overview

CelestiAR is an Augmented Reality (AR)-powered interior design prototype developed using Unity and AR Foundation. The application enables users to visualize furniture placement, room layouts, wallpapers, and flooring designs directly within real-world environments using mobile AR technology.

The project was developed as an academic prototype to explore immersive interior visualization and interactive home decor simulation through AR-based experiences.

---

## 🎯 Objective

The objective of this project is to:

* Enable users to virtually place furniture in real-world spaces
* Improve interior visualization before purchasing furniture or decor
* Provide immersive room customization using AR technology
* Assist users in experimenting with layouts, wallpapers, and flooring options
* Explore practical implementation of AR in interior design applications

## 📱 Application Workflow

1. User launches the application
2. Camera permissions are requested
3. User selects decor category
4. AR camera scans the environment
5. Furniture and decor elements can be placed virtually
6. Users customize layouts and visualize designs in real-time

---

## 🧩 System Design

The application uses AR Foundation and ARCore to detect surfaces and overlay 3D virtual objects within real-world environments.

The workflow includes:

* Camera-based environment scanning
* Plane detection
* 3D model rendering
* Real-time object placement
* User interaction through touch gestures
